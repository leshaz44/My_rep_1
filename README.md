Домашняя работа №1
Эти задания охватывают широкий спектр основных типов данных и структур данных в Python, предлагая как теоретические, так и практические упражнения. Необходимо написать код на Python и оценить сложность вашего алгоритма О(N).
1.	Создайте кортеж, содержащий 4 разных числа. Выведите на экран значение предпоследнего элемента кортежа (используя отрицательный индекс).
2.	Напишите программу, которая принимает строку от пользователя и выводит количество символов в этой строке без учета пробелов.
3.	Создайте словарь, содержащий информацию о студенте: имя, возраст, курс, предметы (список). Выведите все предметы студента через запятую.
4.	Используя цикл, создайте список, содержащий квадраты всех целых чисел от 1 до 10, и выведите его на экран.
5.	Напишите функцию, которая принимает список чисел и возвращает их среднее арифметическое.
6.	Создайте два множества из 4 разных чисел каждое, найдите их пересечение и выведите на экран.
7.	Напишите программу, которая принимает от пользователя целое число и выводит таблицу умножения для этого числа от 1 до 10.
8.	Создайте словарь с пятью парами «ключ-значение», где ключи - названия фруктов, а значения - их цвета. Выведите только те фрукты, цвет которых содержит букву 'к'.
9.	Напишите функцию, которая принимает строку и возвращает True, если строка является палиндромом (игнорируя регистр и пробелы), и False в противном случае.
10.	Создайте список из 5 строк и замените в нем каждый третий символ в каждой строке на звездочку '*'.
11.	Создайте кортеж из 6 элементов разных типов данных и выведите на экран только те элементы, тип которых является числовым (int или float).
12.	Напишите программу, которая принимает от пользователя два числа и выводит результат их деления с обработкой исключения деления на ноль.
13.	Создайте словарь, содержащий информацию о книге (автор, название, год издания, жанр), и выведите информацию в формате: "Название: [название], Автор: [автор], Год: [год], Жанр: [жанр]".
14.	Создайте два множества с названиями городов, найдите города, которые есть в первом множестве, но отсутствуют во втором.
15.	Напишите функцию, которая принимает список чисел и возвращает кортеж из двух элементов: минимального и максимального числа.
16.	Создайте список чисел от 1 до 20, а затем создайте новый список, содержащий числа, которые делятся и на 2, и на 3 одновременно.
17.	Напишите программу, которая принимает от пользователя строку и выводит количество гласных букв в ней.
18.	Создайте кортеж, содержащий 4 элемента различных типов данных, и распакуйте первые три элемента в отдельные переменные.
19.	Напишите функцию, которая принимает список словарей с информацией о студентах и выводит информацию о всех студентах в формате "Имя: [имя], Возраст: [возраст], Курс: [курс]".
20.	Создайте список и добавьте в него 5 различных элементов разных типов данных. Выведите на экран тип данных каждого элемента списка
